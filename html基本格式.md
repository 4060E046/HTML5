html基本格式

```html
<!DOCTYPE html>
<html>
<head>
    <title>網頁標題</title>
    <meta charset="utf-8">
</head>
<body>

</body>
</html>
```
```
body 網頁主體

background=”” //背景圖片 
bgcolor=”” //背景顏色 
text=”” //本文文字顏色

br 強制換行

p 文字分段

align=”right” 
align=”center” 
align=”left” 

pre 原格式
hr 分隔線
h1~h6 文字標題
b 粗體字
i 斜體字
u 文字加底線
sup 上標籤
sub 下標籤
font 設定文字外觀
size=”” //字體大小 
face=”” //文字字型 
color=”” //顏色
```
```
符號	語法
<	&lt
>	&gt
&	&amp
“	&quot
&nbsp	不換行的空白
```
